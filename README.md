# 🚀 ABOUT BOT

**BIHAR KE BAHUBALI WhatsApp Bot** is a **next-level hacker style multi-device WhatsApp automation bot** using **Baileys Library**.  

It lets admins **fully control, protect, and play with WhatsApp groups** while having fun with games, TTS, stickers, and more.  

**Main Features:**  

- 🔥 Tag all group members  
- 🛡️ Anti-link & spam protection  
- 🎮 Games like Tic Tac Toe  
- 🎨 Sticker & TTS generation  
- 🔒 Admin-only commands  
- 🌐 Multi-device support  

---

# ⚡ BOT COMMANDS

| Command | Function |
|--------|----------|
| `.tagall` | Tag all members in group |
| `.kick` | Remove a member |
| `.warn` | Warn a member |
| `.mute` | Mute group |
| `.unmute` | Unmute group |
| `.sticker` | Create a sticker |
| `.tts` | Convert text to speech |
| `.ttt` | Tic Tac Toe game |
| `.menu` | Show all commands |

> 💀 **Hacker Style:** Commands highlighted with emojis & color coded for pro vibes
