## 🚀 ABOUT BOT
**BIHAR KE BAHUBALI Bot** is a **next-level multi-device WhatsApp automation bot** using **Baileys Library**.  

Packed with **hacker-style commands, games, admin tools, anti-spam features**, aur full **fun utilities**.  

**Main Features:**  
- 🔥 `.tagall` → Sab members ko ek saath tag karo  
- 🛡️ Anti-link & spam protection  
- 🎮 Games like Tic Tac Toe `.ttt`  
- 🎨 Stickers & TTS `.sticker / .tts`  
- 🔒 Admin commands `.kick / .warn / .mute / .unmute`  
- 🌐 Multi-device support  
- ⚡ `.menu` → Full command list  

---

## ⚡ COMMANDS (Bihar Style)

| Command | Function |
|--------|----------|
| `.tagall` | Sab members ko tag karo |
| `.kick` | Member ko nikal do |
| `.warn` | Member ko warn karo |
| `.mute` | Group ko mute karo |
| `.unmute` | Group ka mute khol do |
| `.sticker` | Sticker banao |
| `.tts` | Text to speech |
| `.ttt` | Tic Tac Toe game |
| `.menu` | Commands list dikhao |
